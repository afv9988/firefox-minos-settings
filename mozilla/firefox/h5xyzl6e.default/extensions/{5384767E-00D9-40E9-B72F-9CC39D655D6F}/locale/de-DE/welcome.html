Willkommen bei EPUBReader!
So funktioniert es
  • Gehen Sie zu der Website, auf der Sie die ePub-Datei herunterladen wollen und klicken auf den "Download"-Link. Lokal gespeicherte ePub-Dateien können Sie über das Firefox-Menü "Datei/Datei öffnen" öffnen oder Sie ziehen die Datei einfach auf das Firefox-Fenster.
  • Jetzt beginnt EPUBReader zu arbeiten: er lädt die ePub-Datei herunter bzw. öffnet sie, verarbeitet sie und zeigt sie direkt lesefertig an!
  • EPUBReader erstellt eine Seite, auf der alle ePub-Dateien, die Sie heruntergeladen/geöffnet haben, aufgelistet werden. Sie können diese Seite auf vier Wegen öffnen:
    • EPUBReader hat ein Lesezeichen mit dem Namen "ePub-Katalog" an das Ende Ihrer Lesezeichenliste eingefügt.
    • Sie finden ein neues Menü "ePub-Katalog" in dem Firefox-Menü "Extras".
    • Sie können einen Button in die Firefox-Symbolleiste einfügen. Ab Firefox 4.0 wird der Button automatisch hinzugefügt.
    • Wenn Sie eine ePub-Datei lesen, finden Sie unten einen Button, mit dem Sie den ePub-Katalog öffnen können.
Wenn Sie mehr Informationen über EPUBReader haben möchten, lesen Sie bitte das Handbuch oder die FAQs.
